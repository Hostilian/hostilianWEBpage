Kanye REST API

About the Project

This project utilizes the Kanye REST API to fetch random quotes from Kanye West. It is built using Cloudflare Workers, providing a fast and reliable way to access quotes.

Features

  • Fetch random quotes from Kanye West.
  • Simple and intuitive interface.
  • Fast response times thanks to Cloudflare Workers.

Usage

To use the API, simply make a GET request to https://api.kanye.rest. You will receive a random quote in response.
